Nothing about building Catch was really difficult to pull off. I think the hardest thing for me to do though was getting the player to catch the falling balls and getting points from it, if only because Unity collision can be a bit weird sometimes.

The design changed really early actually. I was originally going to make it fully 3D, but then realized that wouldn't work so everything was restricted to 2D to make things much easier for everyone involved.

Some of the scripts do go to other scripts to get things working. So it is possible to change one specific thing and that could result in something else being changed and that will be a whole thing. The code is simple enough though that the effect would probably be minimal.

I think the game is in a good state, so the next thing I would want to improve would be audio. Because the game is completely silent so some sounds could help with the game.
